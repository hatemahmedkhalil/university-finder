## How to Verify and Import

1. Visit the official URL listed for each university.
2. Locate the tuition fees page for **international/non-EU students**.
3. Confirm the fee amount in EUR for the current academic year.
4. Add the university via the UniFind Admin Panel at http://localhost:5174/universities/create
5. Fill in all fields; use the **Admin Notes** field to record the source URL and date verified.
